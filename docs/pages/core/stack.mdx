## Browser support

Stack uses [flexbox gap](https://caniuse.com/flexbox-gap) to add spacing between children.
In older browsers `Stack` children will not have spacing. You can install PostCSS [flex-gap-polyfill](https://github.com/gavinmcfarland/flex-gap-polyfill)
to add support for older browsers.
